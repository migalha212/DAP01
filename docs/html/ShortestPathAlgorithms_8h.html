Computes the shortest paths from a given origin vertex to all other vertices in the graph using Dijkstra's algorithm.

Parameters
gPointer to the graph on which the algorithm will be executed.
originPointer to the origin vertex from which the shortest paths will be calculated.
distanceDecides between using Driving edges or Walking edges.

O((E + V) log |V|) time complexity, O(|V|) space complexity.

Note
The graph should not contain negative weight edges, as Dijkstra's algorithm does not handle them correctly.
